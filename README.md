# Splitzy

Split restaurant bills with friends. Upload a receipt photo, assign items to people, and get the exact amount each person owes — including their share of tax and tip.

## How It Works

1. Upload a photo of your receipt
2. The app extracts line items, tax, and tip using in-browser OCR
3. Add your friends and assign items to each person
4. Get a breakdown of what each person owes
5. Share a link so friends can self-select their items

## Tech

- Vanilla HTML/CSS/JavaScript
- Tesseract.js for in-browser OCR
- No backend — runs entirely in the browser
- Deployed on GitHub Pages
